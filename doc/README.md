## 社团和社区

### 社团（开源鸿蒙社）
- 运营和维护社团的正常运转
- 社团活动的策划、组织、执行
- 社团成员的管理、培养、激励
- 社团的宣传、活动、资源的整合、共享
- 社团的技术交流、合作、资源共享
- 社团成员的开源思想的培养，开源思想的体现和开源项目的开发

### 社区（开源社区）
- 创造和提供一个良好的氛围，让更多的人参与到开源的世界中来
- 促进开源项目的开发和维护，推动开源社区的发展
